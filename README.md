# 🎓 AI Counsellor

## 📌 Overview
AI Counsellor is an intelligent study-abroad advisory platform that assists students throughout the university admission process. The platform leverages AI to provide personalized guidance, university recommendations, profile evaluation, and application tracking.

## 🚀 Features
- AI-powered counselling assistant
- Personalized university recommendations
- Student profile analysis
- Application tracking system
- University comparison
- Task and deadline management
- Dynamic match scoring

## 🛠️ Technologies Used

### Frontend
- React.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- Python

### Database
- PostgreSQL

### AI Integration
- Groq LLM
- Prompt Engineering

### Student Profile Management

### University Recommendation Engine
